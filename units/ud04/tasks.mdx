### Task 4: Linux Process Management

Modify and complete these scripts:

<Accordion title="Exercise 1: Kill Process">
Modify the script to send SIGKILL (signal 9):

```bash
#!/bin/bash

echo " Obtenemos el PID"
PID_AUX=$(ps -e | grep firefox | cut -d " " -f2)
echo " - PID : $PID_AUX"
kill -10 $PID_AUX
exit 0
```

**Task:** Change `-10` to `-9` to send SIGKILL.
</Accordion>

<Accordion title="Exercise 2: Process Tree with PIDs">
Modify to show PIDs in the process tree:

```bash
#!/bin/bash

echo " Obtenemos el PID"
PID_AUX=$(ps -e | grep firefox | cut -d " " -f2)
echo " - PID : $PID_AUX"
pstree $PID_AUX
exit 0
```

**Task:** Add the `-p` parameter to pstree to show PIDs.
</Accordion>

### Task 6: Linux Network Configuration

<Accordion title="Exercise 1: netplan Configuration">
**Objective:** Configure static IP using netplan

<Steps>
  <Step title="Backup Current Config">
    ```bash
    sudo cp /etc/netplan/01-netcfg.yaml /etc/netplan/01-netcfg.yaml.backup
    ```
  </Step>
  
  <Step title="Edit Configuration">
    Create a static IP configuration:
    - IP: 192.168.1.100/24
    - Gateway: 192.168.1.1
    - DNS: 8.8.8.8, 8.8.4.4
  </Step>
  
  <Step title="Apply Configuration">
    ```bash
    sudo netplan try
    ```
  </Step>
  
  <Step title="Verify">
    ```bash
    ip addr
    ip route
    ```
  </Step>
</Steps>

<Warning>
Use `netplan try` - it auto-reverts if you lose connection!
</Warning>
</Accordion>
