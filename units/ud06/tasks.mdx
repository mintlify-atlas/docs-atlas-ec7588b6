## Linux File Operations

### Task 3: Using find Command

<Accordion title="Exercise: Explain find Commands">
**Objective:** Understand the find command

Describe what each find command does:

```bash
# 1
find /dev/ -iname sda*

# 2
find /lib -name libc* -type f

# 3
find /usr/share -type l 

# 4
find /tmp -user $USER

# 5
find /usr/bin/ -type f -executable

# 6
find /var/log/ -daystart -ctime -1

# 7 (this is difficult)
find /boot/ -follow -lname "*"

# 8
find /tmp/ -mtime +31 -delete

# 9
find ./ -size +100M
```

**Research:**
- What does `-iname` do?
- What's the difference between `-name` and `-iname`?
- What is `-type f` vs `-type l`?
- What does `-mtime +31` mean?
- Why is command #8 dangerous?
</Accordion>

### Task 7: Links and Shortcuts

<Accordion title="Windows: Creating Links">
**Objective:** Create symbolic links and shortcuts in Windows

<Steps>
  <Step title="Create Symbolic Link">
    ```powershell
    # Requires Administrator
    New-Item -ItemType SymbolicLink -Path C:\Link -Target C:\Original
    ```
  </Step>
  
  <Step title="Create Hard Link">
    ```powershell
    New-Item -ItemType HardLink -Path C:\HardLink.txt -Target C:\Original.txt
    ```
  </Step>
  
  <Step title="Create Junction">
    ```powershell
    New-Item -ItemType Junction -Path C:\Junction -Target C:\OriginalFolder
    ```
  </Step>
  
  <Step title="Document Differences">
    Explain:
    - Symbolic link vs hard link vs junction
    - When to use each type
    - Limitations of each
  </Step>
</Steps>
</Accordion>

<Accordion title="Linux: Creating Links">
**Objective:** Create symbolic and hard links in Linux

<Steps>
  <Step title="Create Symbolic Link">
    ```bash
    ln -s /original/path /link/path
    ```
  </Step>
  
  <Step title="Create Hard Link">
    ```bash
    ln /original/file.txt /hardlink/file.txt
    ```
  </Step>
  
  <Step title="Experiment and Document">
    1. Create a file with content
    2. Create both symbolic and hard links to it
    3. Modify original - check links
    4. Delete original - check links
    5. Document behavior differences
  </Step>
  
  <Step title="Research Questions">
    Answer:
    - Can symbolic links cross file systems?
    - Can hard links cross file systems?
    - What happens to hard link when original is deleted?
    - What happens to symbolic link when original is deleted?
  </Step>
</Steps>
</Accordion>
