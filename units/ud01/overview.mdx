# Características de los Sistemas Operativos

Esta unidad presenta los conceptos básicos de sistemas informáticos, sus componentes, y los fundamentos de los sistemas operativos modernos.

## Objetivos de Aprendizaje

Al completar esta unidad, serás capaz de:

<CardGroup cols={2}>
  <Card title="Sistema Informático" icon="computer">
    Comprender la arquitectura completa de un sistema informático y sus componentes
  </Card>
  <Card title="Sistema Operativo" icon="microchip">
    Identificar las funciones y clasificación de los sistemas operativos
  </Card>
  <Card title="Tipos de Software" icon="code">
    Distinguir entre diferentes tipos de aplicaciones y software
  </Card>
  <Card title="Licencias" icon="certificate">
    Conocer los tipos de licencias de software y sus implicaciones
  </Card>
</CardGroup>

## Contenidos de la Unidad

<Steps>
  <Step title="El Sistema Informático">
    Componentes físicos, lógicos y humanos que conforman un sistema informático completo
  </Step>
  
  <Step title="El Sistema Operativo">
    Funciones, objetivos, seguridad y abstracción del hardware
  </Step>
  
  <Step title="Clasificación de Sistemas Operativos">
    Tipos según número de usuarios, tareas, y sistemas distribuidos
  </Step>
  
  <Step title="Tipos de Aplicaciones">
    Software de sistema, programación y aplicaciones
  </Step>
  
  <Step title="Tipos de Licencias">
    GPL, BSD, propietarias, OEM, Retail, VLM y educativas
  </Step>
  
  <Step title="Distribuciones GNU/Linux">
    Historia, filosofía del software libre y principales distribuciones
  </Step>
</Steps>
