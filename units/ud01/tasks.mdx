# Ejercicios de la Unidad 01

Esta sección contiene las tareas prácticas que te ayudarán a consolidar los conocimientos teóricos sobre sistemas operativos, tipos de aplicaciones y licencias.

| Función | Aplicación |
| ------- | ---------- |
| Editor de texto sencillo | Notepad (Bloc de Notas)|
| Navegador Web ||

| Cliente de Correo||
| Compilador de C||

<Note>
  Investiga diferentes opciones para cada categoría y selecciona la que consideres más adecuada.
</Note>

### Ejercicio 2: Alternativas de Software

Ya has visto que tenemos multitud de aplicaciones. Sin embargo, muchas veces buscamos **alternativas** a aplicaciones instaladas que no ofrecen lo necesario o se han quedado obsoletas.

Rellena la misma tabla del ejercicio anterior, pero utilizando [AlternativeTo.net](https://alternativeto.net/) para encontrar alternativas.

<Warning>
  Explora las opciones de filtrado por sistema operativo, licencia y popularidad en AlternativeTo.
</Warning>
