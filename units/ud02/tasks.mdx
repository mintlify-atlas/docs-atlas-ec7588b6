### Tarea Extra 2: Primer Contenedor Docker

Si tienes Docker instalado, crea tu primer contenedor:

<CodeGroup>
```bash Paso 1: Ejecutar Ubuntu
docker run -it ubuntu bash
```

```bash Paso 2: Dentro del Contenedor
# Actualizar paquetes
apt update

# Instalar una aplicación
apt install cowsay

# Ejecutar
/usr/games/cowsay "Hello Docker!"
```

```bash Paso 3: Salir y Limpiar
# Salir del contenedor
exit

# Listar contenedores
docker ps -a

# Eliminar contenedor
docker rm <container_id>
```
</CodeGroup>
